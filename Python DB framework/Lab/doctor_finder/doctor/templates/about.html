About MediCare Plus

MediCare Plus is a Django-based Doctor Finder project created for Python DB and Framework assignment. It helps users view doctors, book appointments, login, and make payment using Razorpay test mode.

This project uses Django MVT architecture, Bootstrap UI, database models, ORM CRUD, admin customization, authentication, AJAX, and maps.
